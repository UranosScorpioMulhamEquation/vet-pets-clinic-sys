# vet-pets-clinic-sys
Veterinary Pets Clinic Management System (VPCMS)
نظام إدارة عيادة الحيوانات الأليفة البيطرية
​نبذة عن البرنامج
​نظام متطور ومخصص لإدارة العيادات البيطرية، صُمم خصيصاً لتسهيل سير العمل اليومي للأطباء البيطريين. يجمع البرنامج بين كفاءة قواعد البيانات وسهولة واجهة المستخدم، مما يسمح بإدارة بيانات المرضى (الحيوانات)، الزيارات، الخدمات المالية، والتقارير في مكان واحد.
​المميزات الرئيسية
​نظام أمان متطور: حماية البرنامج عبر "رقم تعريف الجهاز" (Machine ID) لضمان خصوصية بيانات العيادة.
​إدارة المعاملات: إنشاء رقم معاملة فريد لكل زيارة لضمان دقة السجلات.
​المرونة المالية: نظام تسعير مرن يتيح للطبيب تحديد رسوم كل خدمة على حدة مع حساب تلقائي للضريبة (5%).
​قاعدة بيانات قوية: حفظ وتخزين دائم لبيانات العملاء، وتاريخ الحيوانات، والخدمات المقدمة باستخدام SQLite.
​البحث السريع: إمكانية الوصول إلى تاريخ أي عميل من خلال رقم الهاتف فقط.
​واجهة مستخدم محسنة: تصميم عصري يدعم اللغة العربية (RTL) بالكامل لضمان تجربة استخدام مريحة.
​كيفية الاستخدام
​شاشة الحماية: عند التشغيل لأول مرة، سيطلب النظام كلمة مرور. يرجى تزويد المبرمج بـ "رقم تعريف الجهاز" الظاهر على الشاشة للحصول على مفتاح التفعيل.
​إدخال بيانات جديدة:
​انقر على زر "معاملة جديدة".
​قم بتعبئة بيانات الطبيب والحيوان والعميل.
​اختر الخدمات المطلوبة من القائمة المنسدلة.
​اضغط على "إظهار رسوم الخدمات" لإدخال أسعار الخدمات المختارة.
​سيقوم النظام بحساب الإجمالي مع الضريبة تلقائياً قبل الحفظ.
​البحث والتقارير: انتقل لتبويب "بحث وطباعة"، أدخل رقم هاتف العميل، وستظهر لك كافة سجلاته السابقة بشكل منظم.
​المتطلبات التقنية
​هذا البرنامج مبني باستخدام:
​Python 3.x
​Streamlit (لبناء واجهة المستخدم التفاعلية)
​SQLite (قاعدة بيانات خفيفة وقوية)
​Pandas (لمعالجة البيانات وإدارة الجداول)
